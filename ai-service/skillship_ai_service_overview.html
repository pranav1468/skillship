🔌 API Endpoints

GET
/healthz
Health check — returns service status + DB connectivity. No auth required.
POST
/api/tutor/ask
RAG-grounded tutor. Retrieves top-5 course chunks → Gemini answers with citations. Requires pgvector DB.
POST
/api/career/ask
CareerPilot agent. Personalised career path suggestions for Indian students using student profile + Gemini.
POST
/api/quiz/generate
Generates MCQ / TF / SHORT questions for any topic, grade, and difficulty using Gemini JSON mode.
POST
/api/quiz/adaptive-next
IRT-lite engine computes next difficulty from attempt history, then generates one question at that level.
POST
/api/quiz/grade-short
LLM-grades SHORT-answer responses against a rubric. Returns score (0–1) + one-line feedback.
POST
/api/content/tag
Auto-tags uploaded content with topics, grade level, difficulty, and prerequisites using Gemini.
POST
/api/reports/weekly
Analyst agent generates a principal-facing weekly report with highlights, concerns, and recommendations.
POST
/api/risk/scan
Hybrid risk scanner: hard rules (attendance, score drop, logins) flag students, LLM adds nuance + reasoning.

🔍 RAG Pipeline (Tutor)

1
Chunk
rag/chunker.py
PDF or article text split into 800-char overlapping chunks (100-char overlap), word-boundary aware.
2
Embed + Upsert
rag/embedder.py
Each chunk embedded via Gemini text-embedding-004 (768-dim). Upserted into content_chunks table, scoped to school_id + content_id.
3
Retrieve
rag/retriever.py
Student question embedded with task_type=RETRIEVAL_QUERY. Top-5 chunks fetched via ivfflat cosine index, filtered to school_id (tenant isolation).
4
Generate Answer
agents/tutor_agent.py
Chunks injected into Gemini context. System prompt (tutor.md) enforces citation and no direct quiz answers. Returns answer + references.

⚠️ Risk Agent — Hybrid Logic

RuleThresholdFlag
Attendance below 75% attendance
Avg Score (critical) below 40% failing
Avg Score (low) below 55% failing
Score trend last 3 avg < prev 3 avg − 10% score_trending_down
Engagement 0 logins this week disengagement

Flagged students are batched into a single Gemini call that assigns level (low/medium/high/critical), kind, reason, and evidence.
